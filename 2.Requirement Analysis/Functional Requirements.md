# Functional Requirements

The Human Development Index (HDI) Prediction System should provide the following functionalities:

## Home Page

* Display the project introduction.
* Provide navigation to the prediction page.

## Prediction Module

* Accept user inputs:

  * Country
  * Life Expectancy
  * Expected Years of Schooling
  * Mean Years of Schooling
  * Gross National Income (GNI) per Capita
* Validate all input values.
* Send the input data to the trained Machine Learning model.

## Prediction Result

* Display the predicted HDI score.
* Display the corresponding Human Development category:

  * Very High Human Development
  * High Human Development
  * Medium Human Development
  * Low Human Development

## Navigation

* Allow users to return to the prediction page for another prediction.
