# Project Summary

## Project Title

Human Development Index (HDI) Prediction using Machine Learning

## Project Description

The Human Development Index (HDI) Prediction System is a Machine Learning-based web application developed to predict a country's Human Development Index (HDI) score using important socio-economic indicators.

The application accepts the following input parameters:

* Country
* Life Expectancy
* Expected Years of Schooling
* Mean Years of Schooling
* Gross National Income (GNI) per Capita

A trained Linear Regression model processes these inputs and predicts the HDI score. The predicted score is then classified into one of the following categories:

* Very High Human Development
* High Human Development
* Medium Human Development
* Low Human Development

The model is integrated into a Flask web application with a simple and user-friendly interface. The project demonstrates the complete Machine Learning lifecycle, including data preprocessing, model training, deployment, and cloud hosting.
