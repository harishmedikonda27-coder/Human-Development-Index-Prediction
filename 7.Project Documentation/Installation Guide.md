## Running the Project Locally

### Step 1: Clone the Repository

```bash
git clone https://github.com/harishmedikonda27-coder/Human-Development-Index-Prediction.git
```

### Step 2: Navigate to the Flask Folder

```bash
cd Flask
```

### Step 3: Install Required Packages

```bash
pip install -r ../requirements.txt
```

### Step 4: Run the Flask Application

```bash
python app.py
```

### Step 5: Open the Application

Open your web browser and visit:

http://127.0.0.1:5000

The Human Development Index Prediction application will now run on your local machine.
