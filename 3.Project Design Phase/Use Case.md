# Use Case

## Actor

User

## Use Cases

* Open the application.
* Read the project overview.
* Navigate to the prediction page.
* Enter HDI indicator values.
* Submit the prediction request.
* View the predicted HDI score.
* View the Human Development category.
* Perform another prediction if required.

## Expected Outcome

The system successfully predicts the Human Development Index (HDI) score based on the user's input values.
