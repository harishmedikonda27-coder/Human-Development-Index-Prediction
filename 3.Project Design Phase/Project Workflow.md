# Project Workflow

The project follows the steps below:

1. Collect the HDI dataset.
2. Perform data preprocessing.
3. Analyze the dataset using visualization techniques.
4. Train the Linear Regression model.
5. Evaluate the model performance.
6. Save the trained model using Pickle.
7. Develop the Flask web application.
8. Integrate the trained model with Flask.
9. Accept user inputs through the web interface.
10. Predict the HDI score.
11. Display the prediction result and Human Development category.
