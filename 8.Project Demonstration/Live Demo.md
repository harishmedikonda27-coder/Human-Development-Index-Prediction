# Live Demo

## Project Title

Human Development Index (HDI) Prediction using Machine Learning

## Live Application

The application has been successfully deployed on Render.

**Demo Link**

https://human-development-index-prediction-r7ha.onrender.com/

## Description

The live application allows users to:

* Open the Home Page.
* Navigate to the Prediction Page.
* Enter Human Development indicators.
* Predict the Human Development Index (HDI) score.
* View the predicted HDI score and Human Development category in real time.
