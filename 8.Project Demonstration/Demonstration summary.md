# Demonstration Summary

## Project Overview

The Human Development Index (HDI) Prediction System is a Machine Learning-based web application developed using Python, Scikit-learn, and Flask.

The application predicts the Human Development Index score based on the following input parameters:

* Country
* Life Expectancy
* Expected Years of Schooling
* Mean Years of Schooling
* Gross National Income (GNI) per Capita

The trained Linear Regression model generates the predicted HDI score and classifies it into one of the following categories:

* Very High Human Development
* High Human Development
* Medium Human Development
* Low Human Development

## Deployment

* Successfully deployed using Render.
* Source code hosted on GitHub.
* Accessible through a web browser without local installation.

## Outcome

The project demonstrates the complete Machine Learning lifecycle, including data preprocessing, model development, Flask integration, cloud deployment, and documentation.
