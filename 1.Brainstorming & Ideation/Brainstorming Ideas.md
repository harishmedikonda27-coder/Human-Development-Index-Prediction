# Brainstorming Ideas

During the ideation phase, several project ideas were explored before selecting the Human Development Index Prediction System.

## Considered Ideas

* Disease Prediction System
* Student Performance Prediction
* House Price Prediction
* Employee Salary Prediction
* Human Development Index Prediction

## Selected Idea

Human Development Index Prediction using Machine Learning.

## Why this Project?

* Solves a real-world socio-economic problem.
* Uses publicly available HDI dataset.
* Demonstrates practical Machine Learning concepts.
* Provides an interactive web application using Flask.
* Helps users understand how health, education, and income influence human development.
